Description

Project: Push-Up Tracker
สามารถนับจำนวนครั้งในการวิดพื้น สามารถวัดชีพจร มีการจับเวลา สามารถใช้ keypad ในการ input จำนวนครั้งหรือเวลาที่ต้องการวิดพื้น มีเสียงเตือนเมื่อครบจำนวนครั้งหรือหมดเวลา สามารถ input ระยะห่างจากเครื่องถึงอกที่ต้องการได้ เพื่อให้เครื่องนับจำนวนครั้งเมื่อถึงระยะนั้น

Input: ข้อมูลจากเซ็นเซอร์ ultra sonic ,เซ็นเซอร์วัดชีพจร และ keypad ในการรับค่าต่างๆ
Output: จำนวนครั้งที่วิดพื้น, เวลา, และอัตราการเต้นของหัวใจ โดยแสดงผ่านจอ LCD

Equipment

Arduino UNO R4 Wifi

ใช้เป็นตัวรันคำสั่งหลัก

16x2 LCD

ใช้ในการแสดงผลข้อมูล

4x4 Keypad

ใช้ในการใส่ตัวเลข และเลือกโหมด

Pulse Sensor

ใช้ในการนับอัตราเต้นของหัวใจ

Ultrasonic Sensor

ใช้ในการวัดระยะ เพื่อนับการวิดพื้น
